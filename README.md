# KTPM
CRUD Sản phẩm sử dụng Spring Boot (REST API) + React (Vite) + MySQL.
1. Công nghệ sử dụng
Backend – Spring Boot
Spring Web
Spring Data JPA
MySQL Connector
Lombok

Frontend – React
React + Vite
Axios
React Router

Database
MySQL 8+

2. Cách run dự án

Chạy Backend
cd backend
mvn spring-boot:run

Chạy Frontend
cd frontend
npm install
npm run dev
